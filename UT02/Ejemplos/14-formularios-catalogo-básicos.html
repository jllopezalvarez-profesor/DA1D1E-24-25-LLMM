Se considera buena práctica colocarlos en listas.

En el atributo "name" se añaden corchetes "[]" al nombre del campo para que PHP, el lenguaje que procesa la petición en el servidor en la página de ejemplo del instituto, reconozca que es un parámetro que puede recibir varios valores.

Si no se usan corchetes, no funcionará correctamente en PHP, pero el resto de lenguajes de programación sí que lo procesarán correctamente.

Lista desplegable

Permite seleccionar una opción de una lista desplegable.

Sólo permite seleccionar un valor.

Lista desplegable multiselección

Permite seleccionar varias opciones de una lista.

Se activa con el atributo "multiple"

El atributo size indica cuantos elementos se muestran en la lista

Para seleccionar múltiples valores se mantiene pulsado "ctrl"

Si el lenguaje de servidor es PHP, hay que usar [] en el nombre

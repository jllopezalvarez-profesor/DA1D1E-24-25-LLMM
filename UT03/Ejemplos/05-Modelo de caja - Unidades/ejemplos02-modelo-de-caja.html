Ejemplos de modelos de caja

Elementos del modelo de caja

Modelo de caja "convencional" (box-sizing: content-box)

            div.content {
            width: 160px;
            height: 80px;
            padding: 20px;
            border: 8px solid red;
            background: yellow;
            }
            .content-box {
            box-sizing: content-box;
            }
Content box
En este caso:
  • Ancho total: 160px + (2 * 20px) + (2 * 8px) = 216px
  • Alto total: 80px + (2 * 20px) + (2 * 8px) = 136px
  • Ancho de la caja de contenido: 160px
  • Alto de la caja de contenido: 80px

Modelo de caja alternativo (box-sizing: border-box)

                        div.content-container {
                            padding: 2em;
                        }

                        div.content {
                            width: 160px;
                            height: 80px;
                            padding: 20px;
                            border: 8px solid red;
                            background: yellow;
                        }

                        .border-box {
                            box-sizing: border-box;
                        }
Border box
En este caso:
  • Ancho total: 160px
  • Alto total: 80px
  • Ancho de la caja de contenido: 160px - (2 * 20px) - (2 * 8px) = 104px
  • Alto de la caja de contenido: 80px - (2 * 20px) - (2 * 8px) = 24px

Display (block, inline, inline-block)

Convitiendo elementos inline en block

.inline-to-block span { display: block; }

Cada una de las siguientes palabras están en un span, que es elemento en línea, pero se convierte a en bloque.

unodostrescuatrocinco

Sin convertir a elemento en bloque se ven así:

unodostrescuatrocinco

Convirtiendo elementos en bloque a elementos en línea

.block-to-inline li { display: inline; }

En este caso tomamos una lista, formada por elementos "li", que son en bloque, y los convertimos a elementos en línea:

  • 1
  • 2
  • 3
  • 4
  • 5

Sin convertir a elementos en línea se ve así:

  • 1
  • 2
  • 3
  • 4
  • 5

Útil para convertir listas de menús de navegación, normalmente "verticales", a listas horizontales.

Dando ancho o alto a elementos en línea (inline-block)

                .inline-block span {
                display: inline-block;
                width: 250px;
                padding: 20px;
                background-color: coral;
                color: brown;
                text-align: center;
                }

En este párrafo hay varias palabras que se han metido dentro de un span para poder dar ancho, márgenes, alineación a la palabra. Se ha puesto un color de fondo para poder identificarlas claramente.

Colapso de márgenes

                .colapse-example .first-p { margin-bottom: 150px; }
                .colapse-example .second-p { margin-top: 100px; }

En este caso tenemos dos párrafos, el primero tiene un margen inferior de 150 píxeles, y el segundo un margen superior de 100 píxeles. La separación entre los dos párrafos no será de 250 píxeles como podríamos esperar. Será de 150, porque los márgenes colapsan y se mantiene sólo el mayor de los dos.

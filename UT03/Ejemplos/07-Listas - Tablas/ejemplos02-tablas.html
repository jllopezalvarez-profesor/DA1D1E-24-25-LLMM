Pie en colspan

Se pueden observar los dos bordes diferenciados, y como hay espacio entre las celdas. Este espacio se puede ampliar con border-spacing.

Tabla por defecto (sin estilos)

table.tabla-bordes {
            border: 1px solid red;
            }
            table.tabla-bordes th,
            table.tabla-bordes td {
            border: 1px solid blue;
            }
            table.colapsado {
            border-collapse: collapse;
            }

Pie en colspan

En este caso los espacios entre bordes desaparecen. Si se usa border-spacing, se ignora, porque solo funciona con bordes no colapsados.

Tabla en la que no se muestran los bordes de las celdas vacías

            empty-cells: hide;
            }

            table.tabla-bordes-no-vacias td {
            border: 1px solid blue;
            }

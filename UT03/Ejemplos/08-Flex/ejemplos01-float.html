Ejemplos de uso de float

Estilos comunes para los ejemplos

Estilos para los elementos que se flotarán, para resetear flotados, y para contenedores.

/* Clase para dar color al borde del contenedor con los elementos flotados */
            .float-container {
            border: 1px solid red;
            background-color: rgb(255, 0, 0, 10%);
            }
            
            /* Colores y padding para los elementos flotados */
            .float-1 {
            padding: 6em 2em;
            background-color: lightgreen;
            }
            
            .float-2 {
            padding: 4em 2em;
            background-color: lightcoral;
            }
            
            .float-3 {
            padding: 3em 2em;
            background-color: lightseagreen;
            }
            
            .float-4 {
            padding: 5em 2em;
            background-color: lightslategray;
            }
            
            /* Clases para finalizar el flotado */
            .clear-left {
            clear: left;
            }
            
            .clear-right {
            clear: right;
            }
            
            .clear-both {
            clear: both;
            }

Estas reglas se usarán en la mayoría de ejeplos de flotado, y se muestran aquí primero para no repetirlos en todos los estilos.

Este ya no se coloca a continuación, porque incluye "clear: both", de forma que se "resetea" el flotado.

En realidad, no es necesario "flotar" todos los elementos. Una vez que se flota un elemento, todos los que aparecen a continuación, se flotarán hasta que se "resetea" el flotado.

Observa como el contenedor, que tiene borde y fondo, no se ve. Está como "colapsado". Este es uno de los efectos de float que resultan difíciles de solucionar.

Flotado de imágenes

En estos ejemplos las imágenes se generan dinámicamente con "https://dummyimage.com". Es un sitio web para generar imágenes de tamaños específicos, y colores específicos.

Basta con usar en el atributo src la dirección "https://dummyimage.com/anchoxalto/color-fondo/color-texto". Ejemplo: "https://dummyimage.com/1280x720/fff/aaa".

Flotado de imagen a la izquierda, dentro de un párrafo de texto

.img-left {
            float: left;
            padding: 0 1em 1em 0;
            }

Este es un ejemplo de una imagen flotada a la izquierda. El texto fluye alrededor de la imagen, ocupando el espacio restante a la izquierda.

Este sería el párrafo en el que se coloca la imagen, y el texto fluye arrededor de la imagen.

Este párrafo, distinto al anterior, también fluye, proque no ha habido un elemento que deshaga el flotado, un "clear:both".

Este párrafo ya no flota, proque tiene una clase con "clear".

Se recomienda usar márgenes en la imagen para que el texto no quede pegado a la imagen. Esto mejora la legibilidad y la estética general del diseño.

Flotado alrededor de una imagen circular

.img-circle {
            float: left;
            width: 200px;
            height: 200px;
            margin: 10px 15px 10px 0;
            border-radius: 50%;
            clip-path: circle(50%);
            shape-outside: circle(50%);
            }

La imagen en realidad es cuadrada, pero se le da forma circular con "clip-path", y se indica que se rodee de texto con "shape-outside".
